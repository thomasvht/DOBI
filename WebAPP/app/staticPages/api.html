Bike

Add new bike /api/bike/add

What?
This adds a new bike to the database. This is only for the bike, the user will not be added.
Method
POST
Url
/api/bike/add
Data
{
    "Owner": "Enter owner id"
}

Add new user to a bike /api/bike/addUser

What?
This adds a new user to a pre-configured bike.
Method
POST
Url
/api/bike/addUser
Data
{
    "user": "Enter owner id",
    "unlockCode": "Unlock id"
}

Remove user from a bike /api/bike/removeUser/:bike_id

What?
This removes a user from a bike.
Method
POST
Url
/api/bike/removeUser/:bike_id

Get bike list by owner /api/bike/byOwner/:owner_id

What?
This gets a list of all the bikes from a specified owner.
Method
GET
Url
/api/bike/addUser

Get bike list by user /api/byUser/:user_id

What?
This gets a list of all the bikes from a specified user.
Method
GET
Url
/api/byUser/:user_id

Get a single bike /api/single/:bike_id

What?
This returns a single bike.
Method
GET
Url
/api/single/:bike_id

Maintenance

Add new maintenance /api/maintenance/add

What?
This adds a new maintenance to the bike.
Method
POST
Url
/api/maintenance/add
Data
{
    "bikeId": "The bike id",
    "mechanic": "Name of the mechanic",
    "description": "The description,
    "price": "The price"
}

Get list of all maintained /api/maintenance/:bike_id

What?
This is a list of all entries
Method
GET
Url
/api/maintenance/:bike_id

Session

Add new session /api/session/add

What?
This adds a new session.
Method
POST
Url
/api/session/add
Data
{
    "owner": "The id of the owner of the bike",
    "user": "The id of the user of the bike",
    "bikeId": "The id of the bike",
    "dateFrom": "The date of the start of the session",
    "dateTo": "The end date of the session"
}
